# DiscordPunishments

## English
DiscordPunishments is a Minecraft Paper/Spigot plugin that connects Discord moderation workflows with in-game punishment commands. Staff can use Discord buttons and modal forms to issue bans, mutes, and kicks without manually typing long console commands.

### Features
- Discord moderation panel with Ban / Mute / Kick actions.
- Modal forms for nickname, duration, and reason.
- Configurable Minecraft commands for each punishment type.
- Admin permission node for setup and management.
- Java 17 + Maven project structure.

### Build
Use Maven to build the plugin package. The generated jar is created in the local build directory and should be uploaded through Releases, not committed to Git.
